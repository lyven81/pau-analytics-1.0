# Step 3 — Review and Confirm

## What This Step Does

Check the LinkedIn post against a short quality checklist and fix any weak spots before saving the final file.

---

## 3A — Quality Checklist

Go through each item. Fix anything that does not pass before moving on.

### Hook
- [ ] First 1–2 lines work completely on their own before "see more"
- [ ] Hook is counterintuitive, surprising, or challenges a common belief
- [ ] Does not open with a greeting, generic statement, or vague question

### Structure
- [ ] Total word count is 150–250 words
- [ ] Follows the sequence: hook → context → key finding → implication → CTA → hashtags
- [ ] Only one data point is used — not the entire analysis
- [ ] The full answer is NOT given away — reader still needs to click to learn more
- [ ] Blog URL is included and correct
- [ ] 3–5 hashtags only — not more

### Language (English posts)
- [ ] Sentences are short and punchy — no sentence longer than 25 words
- [ ] Active voice throughout
- [ ] No filler phrases: "In today's world", "It is worth noting", "Leverage", "Comprehensive"
- [ ] No em-dashes mid-sentence — use a period or comma instead

### Language (Mandarin posts)
- [ ] Written in 马来西亚华语 — not translated from English sentence by sentence
- [ ] Sentences are short — Mandarin reads better in short bursts on LinkedIn
- [ ] 您 used when addressing the reader
- [ ] Numbers written as numerals (60%, not 百分之六十)
- [ ] Hook still counterintuitive — same standard as English posts

### Tone
- [ ] Reads like a person sharing a genuine insight — not a company pushing a service
- [ ] No mention of pricing or service packages
- [ ] No over-selling or hype language ("game-changing", "revolutionary", "unlock your potential")

---

## 3B — Common Fixes

| Problem | Fix |
|---|---|
| Hook is too vague | Pull a specific number from the blog and lead with it |
| Post gives away too much | Remove the conclusion — stop at the finding, not the answer |
| Post is too long | Cut the context paragraph — go straight from hook to finding |
| CTA feels pushy | Change "Contact us" to "Read the full breakdown here:" |
| Hashtags too generic | Replace #business with something topic-specific (#reseller, #marginanalysis) |
| Mandarin reads stiffly | Read aloud — rewrite any sentence that sounds like an official notice |
